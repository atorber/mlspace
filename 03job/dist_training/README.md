# 分布式训练

本目录对应训练模块中的分布式训练能力，定义多机多卡训练任务的基础约定。

## 必填输入

- 任务名称
- 集群或调度器标识
- 节点数量
- 每个节点的 GPU 数量
- rendezvous 后端或启动方式
- 数据集版本与 manifest 路径

## 运行约定

- 将启动参数记录到实验 manifest 中。
- 所有 rank 使用统一的运行 ID。
- 区分 rank 本地日志与聚合训练摘要。
- 在配置中明确检查点频率与保留策略。

## 平台边界

本目录关注训练任务在集群或调度系统中的启动与运行约定。
单机训练代码结构归 `job/pytorch_standard/`，模型导出后的上线兼容检查归 `serving/compatibility/`。

## 操作检查清单

1. 确认所有节点的环境保持一致。
2. 在启动前固定数据集版本。
3. 验证任务重启和检查点恢复行为。
4. 在 manifest 中记录有效 world size 和精度模式。
