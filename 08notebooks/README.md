# Notebook 索引

本目录作为横向实践资料目录，按用途归档基于百舸平台开展业务时形成的 notebook、实验样例和操作记录。

## 当前结构

- `job/`：训练实践、训练策略和模型训练示例
- `ai_asset/dataset/`：数据格式转换、数据处理与数据集相关示例
- `serving/`：推理验证与服务测试示例
- `dev_instance/`：开发机、工作台和环境使用示例
- `ai_asset/image/`：镜像制作与镜像相关示例

## 放置规则

当内容主要用于讲解、演示或探索时，优先放在 `notebooks/`。
当内容已经沉淀为稳定规范、模板或脚本时，再提升到对应功能模块的主干目录。
