# 数据集模块

本模块对应百度百舸 AI 计算平台中的 AI 资产 `Dataset` 模块，定义数据集应如何被描述、组织、同步和版本化。

## 范围

- 数据集元数据
- 存储与目录约定
- 数据快照与版本规则
- `raw`、`processed`、`gold` 三层职责边界
- 大规模数据集同步行为边界

## 文件

- `catalog_template.yaml`：受治理数据集的标准元数据模板
- `versioning_guideline.md`：数据版本化与回滚约定
- `directory_layout.md`：标准数据目录分层与归属规则
- `sync/`：数据集同步规范
- `examples/lance/`：贴近数据集模块的 Lance 简单示例

## 预期目标

一个数据集不应依赖口口相传才能被理解。
通过标准记录，团队应能清楚知道它来自哪里、发生过哪些变化、存放在哪里，以及由谁负责。
