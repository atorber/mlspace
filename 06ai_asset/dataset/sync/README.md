# 数据同步规范

本目录对应数据集模块中的同步能力，定义大规模数据集同步工具应具备的行为边界。

## 支持目标

- S3 兼容存储
- BOS
- 在适用场景下支持 Hugging Face 数据集或模型仓库

## 预期能力

- 支持断点续传
- 支持校验和或完整性验证
- 明确记录源端与目标端位置
- 在执行潜在破坏性同步前支持 dry-run
- 支持按前缀、分区或版本进行过滤

## 平台边界

本目录关注数据集在不同存储之间的同步与复制行为。
数据集元数据、目录布局和版本规则仍归 `ai_asset/dataset/` 根目录统一管理。

## 治理规则

同步工具不能掩盖数据集版本信息。
所选源快照和目标位置必须始终可审查，并能够被记录进 manifest。
